\# Product Vision – Chat First AI Coach

\#\# Concepto principal

Luego del login, la pantalla principal debe ser un chat con inteligencia artificial similar a cualquier LLM del mercado (como ChatGPT).

El usuario podrá escribir de forma natural y preguntar cualquier cosa.

La aplicación utilizará Groq como proveedor de IA (ya implementado y funcionando correctamente).

La conversación debe ser natural, continua y contextual.

El rol de la IA será:

experto nutricionista    
entrenador personal  

La IA debe comportarse como un asistente inteligente especializado en fitness y nutrición.

\---

\#\# Estructura de la aplicación

\#\#\# Pantalla principal

Después del login se muestra directamente el chat.

El usuario puede:

hablar libremente    
hacer preguntas    
registrar comidas    
registrar ejercicios    
registrar peso    
recibir recomendaciones  

El chat es el centro de la aplicación.

\---

\#\#\# Botón Profile

Habrá un botón para acceder al perfil del usuario.

El perfil almacenará:

edad    
altura    
peso    
sexo    
nivel de actividad física    
objetivo:

\- perder grasa  
\- ganar músculo  
\- mantener peso

tipo de dieta:

\- estándar  
\- keto  
\- vegetariana  
\- etc

tipo de entrenamiento:

\- gimnasio  
\- calistenia  
\- running  
\- mixto

\---

\#\# Función de la IA

La IA debe poder:

leer los datos guardados en el perfil  
leer el historial del usuario  
leer los registros diarios  
mantener contexto en el tiempo

La IA debe recordar:

qué come el usuario  
qué ejercicios realiza  
cómo evoluciona su peso  
qué objetivo tiene  
qué recomendaciones ha dado antes

\---

\#\# Comportamiento esperado del chat

El usuario debe poder escribir de forma natural:

ejemplos:

"desayuné huevos con pan"  
"hoy caminé 8000 pasos"  
"hice entrenamiento de pecho"  
"peso 82kg"  
"qué debería cenar?"  
"cuántas calorías debo consumir?"  
"voy bien con mi objetivo?"

La IA debe:

interpretar el mensaje  
extraer datos relevantes  
guardar datos estructurados en la base de datos  
actualizar progreso  
responder con recomendaciones personalizadas

\---

\#\# Rol del modelo

La IA debe comportarse como:

nutricionista profesional  
entrenador personal

Debe proporcionar:

análisis de progreso  
recomendaciones de dieta  
recomendaciones de entrenamiento  
ajustes de calorías  
feedback motivacional  
alertas de desviación del objetivo

\---

\#\# Memoria del usuario

La IA debe usar como contexto:

perfil del usuario  
objetivo  
historial de comidas  
historial de ejercicios  
historial de peso  
resumen diario  
progreso semanal

La conversación debe mejorar con el tiempo a medida que se acumulan datos.

\---

\#\# Objetivo final

Crear una aplicación donde el usuario solo necesite hablar con el chat.

El chat debe:

registrar información automáticamente  
analizar progreso automáticamente  
dar recomendaciones inteligentes  
adaptarse al usuario

El chat sustituye la mayoría de formularios tradicionales.
